# gestion-proyectos-backend
Backend con API REST para el portal web Gestión de Proyectos

### Clonar repositorio desde Eclipse
Para clonar este repositorio desde Eclipse ir a File > Import > Git > Proyects from Git (with smart import)

### Instalar depencias
Para instalar dependencias click derecho en el proyecto Mavan > Updete Proyect... > Marcar la casilla "Force Update..." > OK
